Scripture Highlighter turns your Bible verses into a clickable link that will open a pop-up with the full scripture reference.

== Description ==

This plugin provides a simple way to utilize the Bibles.org Highlighter tool. Once you install Scripture Highlighter there is no configuration required, however you will see some options to enable or disable auto highlighting and also the ability to choose your default version of the Bible.

== Installation ==

Installing Scripture Highlighter is really easy:

1. In your WordPress admin, go to 'Plugins' > 'Add New'.
2. Search for 'Scripture Highlighter'
3. Find the plugin titled 'Scripture Highlighter' in the list.
3. Click 'Install', then 'Install Now', and then 'Activate Plugin'.

Manual Install

1. Download the plugin.
2. Unzip the plugin to your WordPress plugins directory (/wp-content/plugins/).
3. Activate 'Scripture Highlighter' through the 'Plugins' page in WordPress.
